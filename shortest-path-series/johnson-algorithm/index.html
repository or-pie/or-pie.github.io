Johnson 算法

Johnson 算法通过重新赋权的方法使得 Dijkstra 算法能够解决包含负权重边的有向图上的全节点对最短路径问题。

1. 引言

目前，我们已知有以下方法可以用来解决全节点对最短路径问题。

算法 时间复杂度 算法要求
基于线性表的 Dijkstra 算法（运行 \(\vert V\vert\) 次） \(O(\vert V\vert^3)\) 无负权重边
基于二叉堆的 Dijkstra 算法（运行 \(\vert V\vert\) 次） \(O(\vert V\vert \vert E \vert \log \vert V\vert)\) 无负权重边
基于斐波那契堆的 Dijkstra 算法（运行 \(\vert V\vert\) 次） \(O(\vert V\vert^2 \log \vert V\vert + \vert V\vert \vert E\vert)\) 无负权重边
Bellman-Ford 算法（运行 \(\vert V\vert\) 次） \(O(\vert V\vert^2 \vert E\vert)\)
Floyd-Warshall 算法 \(O(\vert V\vert^3)\)

从时间复杂度方面来看，基于斐波那契堆的 Dijkstra 算法（运行 \(\vert V\vert\) 次）的性能是最好的。在非稠密图上，即 \(|E| = O(|V|^2 \mathbin{/} \log |V|)\) 的情况下，基于二叉堆的 Dijkstra 算法（运行 \(\vert V\vert\) 次）的性能也不亚于 Floyd-Warshall 算法。然而，Dijkstra 算法的使用有严格的前提条件，那就是输入图不能包含负权重边。有没有一种方法可以对图进行预处理，使得 Dijkstra 算法能够顺利执行呢？

在本文，我们介绍可以在 \(O(|V| \log |V| + |V| |E|)\) 的时间内找到所有节点对之间的最短路径的 Johnson 算法，该算法通过重新赋权的方法使得 Dijkstra 算法能够解决包含负权重边的有向图上的全节点对最短路径问题。

2. 算法思想

Johnson 算法使用的技巧称为 重新赋权，其工作原理如下：如果图 \(G\) 中所有的边权重都是非负值，我们可以通过对每个节点运行一次 Dijkstra 算法来找到所有节点对之间的最短路径；如果图 \(G\) 包含负权重边但不包含负权重环，那么只要为每条边 \((u, v) \in E\) 计算出新的非负权重值 \(\hat{w}(u, v)\)，然后使用新权重值运行 Dijkstra 算法即可。新权重函数 \(\hat{w}\) 必须满足下面两个重要条件：

  1. 对于所有节点对 \((u，v) \in V^2\)，一条路径 \(p\) 是在使用权重函数 \(w\) 时从节点 \(u\) 到结点 \(v\) 的一条最短路径，当且仅当 \(p\) 是在使用权重函数 \(\hat{w}\) 时从 \(u\) 到 \(v\) 的一条最短路径；
  2. 对于所有的边 \((u, v) \in E\)，新权重 \(\hat{w}(u, v)\) 为非负值。

为了加以区分，我们使用 \(\hat{\delta}\) 表示使用权重函数 \(\hat{w}\) 时的最短路径权重。事实上，找到满足上面两个条件的新权重函数并不是一件难事。

为了确保第一个条件成立，对于任意边 \((u, v) \in E\)，我们要求 \(\hat{w}(u, v) = w(u, v) + h(u) - h(v)\)，其中 \(h : V \rightarrow \mathbb{R}\) 为任意将节点映射到实数的 势函数。

重新赋权并不改变最短路径

给定有向图 \(G = (V, E)\) 和权重函数 \(w : E \rightarrow \mathbb{R}\)，设 \(h : V \rightarrow \mathbb{R}\) 为任意函数。对于每条边 \((u, v) \in E\)，定义 \(\hat{w}(u, v) = w(u, v) + h(u) - h(v)\)。设 \(p = \langle v_0, v_1, \dots, v_k\rangle\) 为从节点 \(v_0\) 到节点 \(v_k\) 的任意一条路径，那么 \(p\) 是在使用权重函数 \(w\) 时从节点 \(v_0\) 到节点 \(v_k\) 的一条最短路径，当且仅当 \(p\) 是在使用权重函数 \(\hat{w}\) 时从节点 \(v_0\) 到节点 \(v_k\) 的一条最短路径，即 \(w(p) = \delta(u, v)\) 当且仅当 \(\hat{w}(p) = \hat{\delta}(u, v)\)。而且，图 \(G\) 在使用权重函数 \(w\) 时不包含负权重环当且仅当图 \(G\) 在使用权重函数 \(\hat{w}\) 时也不包含负权重环。

证明

首先，对于从节点 \(v_0\) 到节点 \(v_k\) 的任意路径 \(p\)，我们有

\[\begin{align*} \hat{w}(p) & = \sum_{i=1}^k \hat{w}(v_{i-1}, v_i) \\ & = \sum_{i=1}^k (w(v_{i-1}, v_i) + h(v_{i-1}) - h(v_i)) \\ & = \sum_{i=1}^k w(v_{i-1}, v_i) + h(v_0) - h(v_k) \\ & = w(p) + h(v_0) - h(v_k) \end{align*}\]

因为 \(h(v_0)\) 和 \(h(v_k)\) 不依赖于任何具体的路径，如果从节点 \(v_0\) 到节点 \(v_k\) 的一条路径在使用权重函数 \(w\) 时比另一条路径短，则其在使用权重函数 \(\hat{w}\) 时也比另一条短。因此 \(w(p) = \delta(u, v)\) 当且仅当 \(\hat{w}(p) = \hat{\delta}(u, v)\)。

最后，我们证明图 \(G\) 在使用权重函数 \(w\) 时包含负权重环当且仅当图 \(G\) 在使用权重函数 \(\hat{w}\) 时包含负权重环。考虑任意环 \(c = \langle v_0, v_1, \dots, v_k\rangle\)，其中 \(v_0 = v_k\)。因为 \(\hat{w}(p) = w(p) + h(v_0) - h(v_k)\)，我们有 \(\hat{w}(c) = w(c) + h(v_0) - h(v_k) = w(c)\)。因此，环 \(c\) 在使用权重函数 \(w\) 时为负值当且仅当环 \(c\) 在使用权重函数 \(\hat{w}\) 时也为负值。

为了确保第二个条件成立，我们要求势函数 \(h\) 必须满足 三角形不等式，即对于任意边 \((u, v) \in E\) 都有 \(h(v) \le h(u) + w(u, v)\)。

通过重新赋值来生成负权重边

给定有向图 \(G = (V, E)\) 和权重函数 \(w : E \rightarrow \mathbb{R}\)，设 \(h : V \rightarrow \mathbb{R}\) 为满足三角形不等式的函数。对于每条边 \((u, v) \in E\)，设置新权重 \(\hat{w}(u, v) = w(u, v) + h(u) - h(v)\)，可以保证 \(\hat{w}(u, v) \ge 0\)。

综上所述，重新赋权的关键就是找到一个满足三角形不等式的势函数。在负权重环问题中，我们介绍了一种基于虚拟节点的负权重环检测方法，该方法同样可以用于寻找满足三角形不等式的势函数。

如何找到一个满足三角形不等式的势函数？

给定有向图 \(G = (V, E)\) 和权重函数 \(w : E \rightarrow \mathbb{R}\)，我们制作一幅扩展图 \(G' = (V', E')\)，其中 \(V' = V \cup \{s\}\)，\(s \not\in V\) 是一个新的虚拟节点，\(E' = E \cup \{(s, v) : v \in V\}\)，并且对权重函数 \(w\) 进行扩展，使得对于所有节点 \(v \in V\)，\(w(s, v) = 0\)。注意，因为虚拟节点 \(s\) 没有进入的边，除了以 \(s\) 为源点的最短路径外，图 \(G'\) 中没有其他包含节点 \(s\) 的最短路径。而且，图 \(G'\) 不包含负权重环当且仅当图 \(G\) 不包含负权重环。现在假定图 \(G\) 和图 \(G'\) 都不包含负权重环。

在扩展图 \(G'\) 上运行 Bellman-Ford 算法，计算虚拟节点 \(s\) 到每个节点 \(v \in V\) 的最短路径权重 \(\delta(s, v)\)，并定义 \(h(v) = \delta(s, v)\)。如果 Bellman-Ford 算法返回 true，根据最短路径的三角形不等式，对于所有的边 \((u, v) \in E\)，有 \(\delta(s, v) \le \delta(s, u) + w(u, v)\)，因此 \(h\) 也满足三角形不等式。如果 Bellman-Ford 算法返回 false，说明输入图中含有负权重环。

在具体实现的时候，我们直接在输入图 \(G\) 上运行全源点 Bellman-Ford 算法即可。

3. Johnson 算法

给定有向图 \(G = (V, E)\) 和权重函数 \(w : E \rightarrow \mathbb{R}\)，Johnson 算法返回一个布尔值以表明输入图中是否存在负权重环。如果存在负权重环，算法将返回 false。否则，算法将返回 true，并且给出所有节点对之间的最短路径和它们的权重。Johnson 算法在运行中需要使用全源点 Bellman-Ford 算法和 Diikstra 算法作为自己的子程序。

下面的代码描述了基于队列版全源点 Bellman-Ford 算法和懒惰 Dijkstra 算法的 Johnson 算法。算法第 2 行运行全源点 Bellman-Ford 算法求解从虚拟节点 \(s\) 到其他节点的使用权重函数 \(w\) 时的最短路径。如果输入图中存在负权重环，算法第 18 行将报告这个问题。算法第 3–16 行假定输入图不包含负权重环。算法第 3–6 行创建一张新的图 \(H = (V, E)\)，为图 \(H\) 中的每条边 \((u, v) \in E\) 赋予新的权重 \(\hat{w}(u, v) = w(u, v) + \delta(s, u) - \delta(s, v)\)。算法第 9–15 行的 for 循环通过调用 \(|V|\) 次 Dijkstra 算法来计算使用权重函数 \(\hat{w}\) 时所有节点对之间的最短路径权重 \(\hat{\delta}(u, v)\)，并将 \(\delta(u, v) = \hat{\delta}(u, v) - \delta(s, u) + \delta(s, v)\) 和对应的前驱信息记录在距离矩阵和前驱矩阵中。

bool johnson(const Graph &G) {
  if (ASBFQ.all_source_bellman_ford_queue(G)) {
    Graph H(G.n);
    for (auto [u, v, w] : G.E) {
      H.add_edge(u, v, w + ASBFQ.dist[u] - ASBFQ.dist[v]);
    }
    dist.assign(G.n, std::vector<double>(G.n));
    prev.assign(G.n, std::vector<int>(G.n));
    for (int u = 0; u < G.n; u++) {
      LD.lazy_dijkstra(H, u);
      for (int v = 0; v < G.n; v++) {
        dist[u][v] = LD.dist[v] - ASBFQ.dist[u] + ASBFQ.dist[v];
        prev[u][v] = LD.prev[v];
      }
    }
    return true;
  } else {
    return false;
  }
}

代码说明

在代码中，ASBFQ 和 LD 表示队列版全源点 Bellman-Ford 算法对象和懒惰 Dijkstra 算法对象。

如果使用斐波那契堆来实现 Dijkstra 算法里面的优先队列，那么 Johnson 算法的总运行时间为 \(O(|V| \log |V| + |V| |E|)\)。上面的代码使用了简单的二叉堆实现，运行时间为 \(O(|V| |E| \log |V|)\)，在稀疏图的情况下，该时间仍然比 Floyd-Warshall 算法的时间表现要好。

#include "johnson.h"

int main() {
  Graph G(5);
  G.add_edge(0, 1, 3);
  G.add_edge(0, 2, 8);
  G.add_edge(0, 4, -4);
  G.add_edge(1, 3, 1);
  G.add_edge(1, 4, 7);
  G.add_edge(2, 1, 4);
  G.add_edge(3, 0, 2);
  G.add_edge(3, 2, -5);
  G.add_edge(4, 3, 6);

  Johnson J;
  if (J.johnson(G)) {
    std::cout << "距离矩阵：" << std::endl;
    for (int i = 0; i < G.n; i++) {
      for (int j = 0; j < G.n; j++) {
        std::cout << std::setw(4) << J.dist[i][j];
      }
      std::cout << std::endl;
    }
    std::cout << "前驱矩阵：" << std::endl;
    for (int i = 0; i < G.n; i++) {
      for (int j = 0; j < G.n; j++) {
        std::cout << std::setw(4) << J.prev[i][j];
      }
      std::cout << std::endl;
    }
  } else {
    std::cout << "存在负权重环：";
    std::vector<int> c = J.reconstruct_negative_weight_cycle();
    for (int i = 0; i < c.size(); i++) {
      std::cout << c[i] << (i < c.size() - 1 ? "->" : "");
    }
    std::cout << std::endl;
  }

  return 0;
}

#ifndef JOHNSON_H
#define JOHNSON_H

#include "all-source-bellman-ford-queue.h"
#include "lazy-dijkstra.h"

class Johnson {
public:
  AllSourceBellmanFordQueue ASBFQ;
  LazyDijkstra LD;

  std::vector<std::vector<double>> dist;
  std::vector<std::vector<int>> prev;

  bool johnson(const Graph &G) {
    if (ASBFQ.all_source_bellman_ford_queue(G)) {
      Graph H(G.n);
      for (auto [u, v, w] : G.E) {
        H.add_edge(u, v, w + ASBFQ.dist[u] - ASBFQ.dist[v]);
      }
      dist.assign(G.n, std::vector<double>(G.n));
      prev.assign(G.n, std::vector<int>(G.n));
      for (int u = 0; u < G.n; u++) {
        LD.lazy_dijkstra(H, u);
        for (int v = 0; v < G.n; v++) {
          dist[u][v] = LD.dist[v] - ASBFQ.dist[u] + ASBFQ.dist[v];
          prev[u][v] = LD.prev[v];
        }
      }
      return true;
    } else {
      return false;
    }
  }

  std::vector<int> reconstruct_shortest_path(int u, int v) {
    if (ASBFQ.clue != nil || dist[u][v] == inf) {
      return {};
    } else {
      std::vector<int> p;
      for (int x = v; x != nil; x = prev[u][x]) {
        p.push_back(x);
      }
      std::reverse(p.begin(), p.end());
      return p;
    }
  }

  std::vector<int> reconstruct_negative_weight_cycle() {
    return ASBFQ.reconstruct_negative_weight_cycle();
  }
};

#endif

#ifndef ALL_SOURCE_BELLMAN_FORD_QUEUE_H
#define ALL_SOURCE_BELLMAN_FORD_QUEUE_H

#include "graph.h"
#include "queue.h"

class AllSourceBellmanFordQueue {
public:
  std::vector<double> dist;
  std::vector<int> prev;
  std::vector<int> rank;
  int clue;

  bool relax(int u, int v, double w) {
    if (dist[v] > dist[u] + w) {
      dist[v] = dist[u] + w;
      prev[v] = u;
      rank[v] = rank[u] + 1;
      return true;
    } else {
      return false;
    }
  }

  bool all_source_bellman_ford_queue(const Graph &G) {
    dist.assign(G.n, 0);
    prev.assign(G.n, nil);
    rank.assign(G.n, 0);
    Queue Q(G.n);
    for (int v = 0; v < G.n; v++) {
      Q.push(v);
    }
    while (not Q.empty()) {
      int u = Q.pop();
      for (auto [v, w] : G.A[u]) {
        if (relax(u, v, w)) {
          if (rank[v] == G.n) {
            clue = v;
            return false;
          }
          if (not Q.contains(v)) {
            Q.push(v);
          }
        }
      }
    }
    clue = nil;
    return true;
  }

  std::vector<int> reconstruct_negative_weight_cycle() {
    if (clue == nil) {
      return {};
    } else {
      std::vector<bool> aux(prev.size(), false);
      std::vector<int> p;
      int x = clue;
      while (not aux[x]) {
        aux[x] = true;
        p.push_back(x);
        x = prev[x];
      }
      std::vector<int> c = {x};
      for (auto it = p.rbegin(); *it != x; it++) {
        c.push_back(*it);
      }
      c.push_back(x);
      return c;
    }
  }
};

#endif

#ifndef QUEUE_H
#define QUEUE_H

#include <bits/stdc++.h>

class Queue {
private:
  std::deque<int> data;
  std::vector<bool> in_queue;

public:
  Queue(int n) : data(), in_queue(n, false) {}

  bool empty() { return data.empty(); }

  bool contains(int v) { return in_queue[v]; }

  void push(int v) {
    data.push_back(v);
    in_queue[v] = true;
  }

  int pop() {
    int v = data.front();
    data.pop_front();
    in_queue[v] = false;
    return v;
  }
};
#endif

#ifndef LAZY_DIJKSTRA_H
#define LAZY_DIJKSTRA_H

#include "graph.h"
#include "priority-queue.h"

class LazyDijkstra {
public:
  std::vector<double> dist;
  std::vector<int> prev;

  bool relax(int u, int v, double w) {
    if (dist[v] > dist[u] + w) {
      dist[v] = dist[u] + w;
      prev[v] = u;
      return true;
    } else {
      return false;
    }
  }

  void lazy_dijkstra(const Graph &G, int s) {
    dist.assign(G.n, inf);
    prev.assign(G.n, nil);
    dist[s] = 0;
    PriorityQueue Q;
    Q.insert(s, 0);
    while (not Q.empty()) {
      auto [u, du] = Q.extract_min();
      if (dist[u] == du) {
        for (auto [v, w] : G.A[u]) {
          if (relax(u, v, w)) {
            Q.insert(v, dist[v]);
          }
        }
      }
    }
  }

  std::vector<int> reconstruct_shortest_path(int v) {
    if (dist[v] == inf) {
      return {};
    } else {
      std::vector<int> p;
      for (int x = v; x != nil; x = prev[x]) {
        p.push_back(x);
      }
      std::reverse(p.begin(), p.end());
      return p;
    }
  }
};

#endif

#ifndef PRIORITY_QUEUE_H
#define PRIORITY_QUEUE_H

#include <bits/stdc++.h>

class PriorityQueue {
private:
  std::priority_queue<std::pair<int, double>,
                      std::vector<std::pair<int, double>>,
                      std::function<bool(std::pair<int, double> &,
                                         std::pair<int, double> &)>> data;

public:
  PriorityQueue() : data([](auto &a, auto &b) { return a.second > b.second; }) {}

  bool empty() { return data.empty(); }

  std::pair<int, double> extract_min() {
    std::pair<int, double> min = data.top();
    data.pop();
    return min;
  }

  void insert(int v, double d) { data.emplace(v, d); }
};

#ifndef GRAPH_H
#define GRAPH_H

#include <bits/stdc++.h>

class Graph {
public:
  int n;
  std::vector<std::tuple<int, int, double>> E;
  std::vector<std::vector<std::pair<int, double>>> A;

  Graph(int n) : n(n), E(), A(n) {}

  void add_edge(int u, int v, double w) {
    E.emplace_back(u, v, w);
    A[u].emplace_back(v, w);
  }

  void clear() {
    E.clear();
    A.assign(n, {});
  }
};
