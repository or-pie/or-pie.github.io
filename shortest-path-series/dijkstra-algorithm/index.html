Dijkstra 算法

Dijkstra 算法是一种解决不含负权重边的有向图上的单源点最短路径问题的算法。

1. 算法思想

Dijkstra 算法解决的是带权重的有向图上的单源点最短路径问题，该算法要求所有边权重都为非负值。因此，在本节的讨论中，给定有向图 \(G = (V, E)\) 和权重函数 \(w : E \rightarrow \mathbb{R}_{\ge 0}\)，对于所有的边 \((u, v) \in E\)，都有 \(w(u, v) \ge 0\)。我们稍后将看到，如果所采用的实现方式合适，Dijkstra 算法的运行时间要低于 Bellman-Ford 算法的运行时间。

首先，Dijkstra 算法采用与 Bellman-Ford 算法相同的初始化操作，所有节点的前驱节点设置为 \(nil\)，源点 \(s\) 的最短路径估计值设置为 \(0\)，其他节点的最短路径估计值设置为 \(\infty\)。算法采用 贪心策略 对节点进行处理，在运行过程中使用 优先队列 来维护所有剩余未处理的节点。我们用 \(Q\) 表示这个优先队列，算法开始的时候它包含所有节点。算法不断从优先队列 \(Q\) 中取出当前最短路径估计值（即 \(dist\) 值）最小的节点 \(u\)，对它发出的每条边 \((u, v) \in A[u]\) 进行松弛操作。如果松弛操作成功降低了 \(dist[v]\)，则需要根据优先队列的具体实现，相应地调整节点 \(v\) 在 \(Q\) 中的位置。

bool relax(int u, int v, double w) {
  if (dist[v] > dist[u] + w) {
    dist[v] = dist[u] + w;
    prev[v] = u;
    return true;
  } else {
    return false;
  }
}

void dijkstra(const Graph &G, int s) {
  dist.assign(G.n, inf);
  prev.assign(G.n, nil);
  dist[s] = 0;
  PriorityQueue Q(G.n);
  for (int v = 0; v < G.n; v++){
    Q.insert(s, dist[s]);
  }
  while (not Q.empty()) {
    int u = Q.extract_min();
    for (auto [v, w] : G.A[u]) {
      if (relax(u, v, w)) {
        Q.decrease_key(v, dist[v]);
      }
    }
  }
}

下图描述的是在有 5 个节点的图上运行 Dijkstra 算法的过程。算法第 12–14 行执行的是 \(dist\) 值和 \(prev\) 值的初始化，第 15–18 行创建一个包含所有节点的优先队列 \(Q\)。在第 19–26 行的 while 循环的每一轮，第 20 行从 \(Q\) 中取出 \(dist\) 值最小的节点 \(u\)。注意，循环第一轮取出的节点必然是源点 \(s\)。然后，第 21–25 行对所有从节点 \(u\) 发出的边 \((u, v) \in A[u]\) 进行松弛操作。如果一条经过节点 \(u\) 的路径能够使得从源点 \(s\) 到节点 \(v\) 的最短路径权重比当前的估计值 \(dist[v]\) 更小，我们就更新 \(dist[v]\) 和 \(prev[v]\)。注意，在算法的第 18 行之后，我们不再向 \(Q\) 插入任何节点，而每个节点都要从 \(Q\) 中被取走一次。因此，算法第 19–26 行的 while 循环的执行次数刚好为 \(|V|\) 轮。

2. 正确性证明

与 Bellman-Ford 算法一样，在 Dijkstra 算法运行过程中，对于任意节点 \(v \in V\)，\(dist[v]\) 总是 \(\delta(s, v)\) 的上界。

上界性质

在 Dijkstra 算法运行过程中，对于所有节点 \(v \in V\)，总是有 \(dist[v] \ge \delta(s, v)\)。

证明

使用归纳法进行证明。

  • 基础步：在初始化之后，对于所有节点 \(v \in V\)，有 \(dist[v] = \infty\)，显然 \(dist[v] \ge \delta(s, v)\)；
  • 归纳步：在对边 \((u, v)\) 进行松弛操作时，唯一可能发生变化的值是 \(dist[v] = dist[u] + w(u, v) \ge \delta(s, u) + w(u, v) \ge \delta(s, v)\)。

下面证明 Dijkstra 算法的正确性。

算法正确性

如果输入图中所有边权重都为非负值，那么 Dijkstra 算法在终止时，对于所有节点 \(v \in V\)，都有 \(dist[v] = \delta(s, v)\)。

证明

使用归纳法进行证明。首先，算法第一轮取出的节点是源点 \(s\)，显然 \(dist[s] = \delta(s, s) = 0\)。考虑算法后续每一轮开始时，对于每个已经处理过的节点 \(v \in V \setminus Q\)，都有 \(dist[v] = \delta(s, v)\)，我们只需要证明此时优先队列 \(Q\) 中 \(dist\) 值最小的节点 \(u\) 满足 \(dist[u] = \delta(s, u)\) 即可。

首先，节点 \(u\) 必然有别于源点 \(s\)，因为源点 \(s\) 已经在第一轮被取出。其次，如果节点 \(u\) 从源点 \(s\) 不可达，那么 \(dist[u] = \delta(s, u) = \infty\)。下面我们集中注意力讨论节点 \(u\) 从源点 \(s\) 可达的情况。令 \(p\) 为从源点 \(s\) 到节点 \(u\) 的一条最短路径，以及 \((x, y)\) 为路径 \(p\) 上第一条满足 \(x \in V \setminus Q\) 和 \(y \in Q\) 的边，于是我们可以将路径 \(p\) 分解为 \(\langle s \stackrel{p_1}{\rightsquigarrow} x \rightarrow y \stackrel{p_2}{\rightsquigarrow} u\rangle\)。注意，子路径 \(p_1\)、\(p_2\) 可能不包含任何边。通过下面的推导过程，我们最终可以得到 \(dist[u] = \delta(s, u)\)：

  1. 因为节点从 \(Q\) 中取出后总是对其发出的边进行松弛操作，所以有 \(dist[y] \le dist[x] + w(x, y) = \delta(s, x) + w(x, y) = \delta(s, y)\)；
  2. 由上界性质得到 \(dist[y] \ge \delta(s, y)\)；
  3. 结合不等式 1 和 2，得到 \(dist[y] = \delta(s, y)\)；
  4. 因为 \(u\) 是 \(Q\) 中 \(dist\) 值最小的节点，所以有 \(dist[u] \le dist[y]\)；
  5. 因为所有边权重都是非负值，所以有 \(\delta(s, y) \le \delta(s, u)\)；
  6. 结合等式 3 和不等式 4、5，得到 \(dist[u] \le dist[y] = \delta(s, y) \le \delta(s, u)\)；
  7. 由上界性质得到 \(dist[u] \ge \delta(s, u)\)；
  8. 结合不等式 6 和 7，得到 \(dist[u] = dist[y] = \delta(s, y) = \delta(s, u)\)。

3. 具体实现

事实上，Dijkstra 算法通常采用 推迟入队 的实现方式，即只在首次到达节点时才将其添加到优先队列，而不是一开始就把所有节点添加进优先队列。这种实现方式的优点是在算法运行过程中，优先队列只需要维护 \(dist\) 值小于 \(\infty\) 的未处理节点。因为对于所以已处理过的节点 \(v \in V \setminus Q\)，都有 \(dist[v] = \delta(s, v)\)，不会有任何后续的松弛操作可以降低 \(dist[v]\)。所以在这种实现方式中，每个节点只会进入优先队列最多一次。

接下来我们探讨几种不同的优先队列数据结构对 Dijkstra 算法性能的影响。在分析时间复杂度的时候，我们不失一般性地假设 \(|E| = \Omega(|V|)\) 和 \(|E| = O(|V|^2)\)，即 \(|E|\) 的数量级不小于 \(|V|\) 的数量级且不大于 \(|V|^2\) 的数量级。Dijkstra 算法一共涉及三种优先队列操作，其运行时间取决于优先队列的具体实现：

  • extract_min 操作取出并返回优先队列 \(Q\) 中 \(dist\) 值最小的节点，共执行 \(O(|V|)\) 次；
  • insert 操作将节点 \(v\) 插入优先队列 \(Q\) 中，共执行 \(O(|V|)\) 次；
  • decrease_key 操作调整节点 \(v\) 在优先队列 \(Q\) 的数据结构中的位置，共执行 \(O(|E|)\) 次。

3.1. 线性表

Dijkstra 算法的一种朴素实现是使用 线性表（如数组、链表等）实现优先队列 \(Q\)：

  • 对于 extract_min 操作，需要使用线性搜索的方式找到并移除 \(dist\) 值最小的节点，运行时间是 \(O(|V|)\)，共执行 \(O(|V|)\) 次；
  • 对于 insert 操作，只需要将节点 \(v\) 附加到线性表一端即可，运行时间是 \(O(1)\)，共执行 \(O(|V|)\) 次；
  • decrease_key 操作的运行时间是 \(O(1)\)，共执行 \(O(|E|)\) 次。

算法总运行时间为 \(O(|V|^2)\)。

3.2. 二叉堆

二叉堆 基于 完全二叉树 实现，是最常见也最容易实现的堆数据结构，其中最底层节点靠左填充，而其他层都被节点填满。作为最堆数据结构，二叉堆满足如下 堆性质：每个节点的值都是以其为根节点的子树中的最小值。基于二叉堆的 Dijkstra 算法时间复杂度分析如下：

  • 对于 extract_min 操作，首先记录堆顶元素（根节点），这是 extract_min 操作的返回值。将堆底元素（底层最右节点）移动并覆盖原堆顶元素，不断向下调整该节点的位置直到堆性质被满足。单次 extract_min 操作的运行时间是 \(O(\log |V|)\)，共执行 \(O(|V|)\) 次；
  • 对于 insert 操作，将节点 \(v\) 附加到堆底，不断向上调整该节点的位置直到堆性质被满足。单次 insert 操作的运行时间是 \(O(\log |V|)\)，共执行 \(O(|V|)\) 次；
  • 对于 decrease_key 操作，首先需要找到节点 \(v\) 当前在堆中的位置，不断向上调整该节点的位置直到堆性质被满足。我们可以引入额外属性实时记录节点 \(v\) 在堆中的位置，那么单次 decrease_key 操作的运行时间是 \(O(\log |V|)\)，共执行 \(O(|E|)\) 次。

算法总运行时间为 \(O(|E| \log |V|)\)。

3.3. 斐波那契堆

事实上，我们可以进一步提高 Dijkstra 算法的运行效率，方法是使用 斐波那契堆 来实现优先队列，因其具有比二叉堆更低的 均摊运行时间1：

  • extract_min 操作的均摊运行时间是 \(O(\log |V|)\)，共执行 \(O(|V|)\) 次；
  • insert 操作的均摊运行时间是 \(O(1)\)，共执行 \(O(|V|)\) 次；
  • decrease_key 操作的均摊运行时间是 \(O(1)\)，共执行 \(O(|E|)\) 次。

算法总运行时间为 \(O(|V| \log |V| + |E|)\)。

3.4. 时间复杂度对比

优先队列数据结构 Dijkstra 算法时间复杂度
线性表 \(O(\vert V\vert^2)\)
二叉堆 \(O(\vert E\vert \log \vert V\vert)\)
斐波那契堆 \(O(\vert V\vert \log \vert V\vert + \vert E\vert)\)

基于斐波那契堆的 Dijkstra 算法拥有最小的理论时间复杂度，但是由于斐波那契堆在实践中有更大的常数因子，在实际应用中往往不如基于二叉堆的 Dijkstra 算法。对于另外两种实现，基于线性表的 Dijkstra 算法和基于二叉堆的 Dijkstra 算法，哪个性能更好呢？比较这两种实现的算法时间复杂度，不难得出如下结论：

  • 当 \(|E| = \omega(|V|^2 \mathbin{/} \log |V|)\) 时，即 \(|E|\) 的数量级大于 \(|V|^2 \mathbin{/} \log |V|\) 的数量级，或者说在 稠密图 上，基于线性表的 Dijkstra 算法性能更好；
  • 当 \(|E| = o(|V|^2 \mathbin{/} \log |V|)\) 时，即 \(|E|\) 的数量级小于 \(|V|^2 \mathbin{/} \log |V|\) 的数量级，或者说在 稀疏图 上，基于二叉堆的 Dijkstra 算法性能更好；
  • 当 \(|E| = \Theta(|V|^2 \mathbin{/} \log |V|)\) 时，即 \(|E|\) 的数量级与 \(|V|^2 \mathbin{/} \log |V|\) 的数量级相当，一般认为基于二叉堆的 Dijkstra 算法性能更好，因为（在运气好的情况下）算法运行过程中 relax 操作的成功次数可能远小于边的数量 \(|E|\)。

#ifndef DIJKSTRA_H
#define DIJKSTRA_H

#include "graph.h"

template <typename PriorityQueue> class Dijkstra {
public:
  std::vector<double> dist;
  std::vector<int> prev;

  bool relax(int u, int v, double w) {
    if (dist[v] > dist[u] + w) {
      dist[v] = dist[u] + w;
      prev[v] = u;
      return true;
    } else {
      return false;
    }
  }

  void dijkstra(const Graph &G, int s) {
    dist.assign(G.n, inf);
    prev.assign(G.n, nil);
    dist[s] = 0;
    PriorityQueue Q(G.n);
    Q.insert(s, 0);
    while (not Q.empty()) {
      int u = Q.extract_min();
      for (auto [v, w] : G.A[u]) {
        if (relax(u, v, w)) {
          if (not Q.contains(v)) {
            Q.insert(v, dist[v]);
          } else {
            Q.decrease_key(v, dist[v]);
          }
        }
      }
    }
  }

  std::vector<int> reconstruct_shortest_path(int v) {
    if (dist[v] == inf) {
      return {};
    } else {
      std::vector<int> p;
      for (int x = v; x != nil; x = prev[x]) {
        p.push_back(x);
      }
      std::reverse(p.begin(), p.end());
      return p;
    }
  }
};

#endif

#ifndef LINEAR_LIST_H
#define LINEAR_LIST_H

#include <bits/stdc++.h>

class LinearList {
private:
  std::vector<std::pair<int, double>> data;
  std::vector<int> index;

public:
  LinearList(int n) : data(), index(n, -1) {}

  bool empty() { return data.empty(); }

  bool contains(int v) { return index[v] != -1; }

  int extract_min() {
    auto it = std::min_element(data.begin(), data.end(), [](auto &a, auto &b) {
      return a.second < b.second;
    });
    int v = it->first;
    if (int i = index[v]; i != data.size() - 1) {
      data[i] = data.back();
      index[data[i].first] = i;
    }
    data.pop_back();
    index[v] = -1;
    return v;
  }

  void insert(int v, double d) {
    data.emplace_back(v, d);
    index[v] = data.size() - 1;
  }

  void decrease_key(int v, double d) { data[index[v]].second = d; }
};

#endif

#ifndef BINARY_HEAP_H
#define BINARY_HEAP_H

#include <bits/stdc++.h>

class BinaryHeap {
private:
  std::vector<std::pair<int, double>> data;
  std::vector<int> index;

  void sift_up(int i) {
    std::pair<int, double> item = data[i];
    for (int j; i > 0 && item.second < data[j = (i - 1) / 2].second;) {
      data[i] = data[j];
      index[data[i].first] = i;
      i = j;
    }
    data[i] = item;
  }

  int smaller_child(int i) {
    if (i * 2 + 2 == data.size() ||
        data[i * 2 + 1].second < data[i * 2 + 2].second) {
      return i * 2 + 1;
    } else {
      return i * 2 + 2;
    }
  }

  void sift_down() {
    int i = 0;
    std::pair<int, double> item = data[i];
    for (int j; i * 2 + 1 < data.size() &&
                item.second > data[j = smaller_child(i)].second;) {
      data[i] = data[j];
      index[data[i].first] = i;
      i = j;
    }
    data[i] = item;
  }

public:
  BinaryHeap(int n) : data(), index(n, -1) {}

  bool empty() { return data.empty(); }

  bool contains(int v) { return index[v] != -1; }

  int extract_min() {
    int v = data[0].first;
    index[v] = -1;
    if (data.size() == 1) {
      data.pop_back();
    } else {
      data[0] = data.back();
      index[data[0].first] = 0;
      data.pop_back();
      sift_down();
    }
    return v;
  }

  void insert(int v, double d) {
    data.emplace_back(v, d);
    sift_up(index[v] = data.size() - 1);
  }

  void decrease_key(int v, int d) {
    data[index[v]].second = d;
    sift_up(index[v]);
  }
};

#ifndef FIBONACCI_HEAP_H
#define FIBONACCI_HEAP_H

#include <bits/stdc++.h>

class FibonacciHeap {
private:
  class Node {
  public:
    int first;
    double second;
    int degree;
    bool mark;
    Node *parent;
    Node *child;
    Node *left;
    Node *right;

    Node(int v, double d)
        : first(v), second(d), degree(0), mark(false),
          parent(nullptr), child(nullptr), left(nullptr), right(nullptr) {}
  };

  Node *root;
  std::vector<Node *> index;
  std::vector<Node *> aux;

  void link(Node *node, Node *parent) {
    node->parent = parent;
    if (parent != nullptr) {
      parent->degree++;
    }
    Node *&base = parent == nullptr ? root : parent->child;
    if (base == nullptr) {
      node->left = node;
      node->right = node;
      base = node;
    } else {
      node->left = base;
      node->right = base->right;
      base->right->left = node;
      base->right = node;
    }
  }

  void insert(Node *node) {
    node->mark = false;
    link(node, nullptr);
    if (root != node && root->second > node->second) {
      root = node;
    }
  }

  void unlink(Node *node) {
    Node *parent = node->parent;
    if (parent != nullptr) {
      parent->degree--;
    }
    Node *&base = parent == nullptr ? root : parent->child;
    if (base == base->right) {
      base = nullptr;
    } else {
      node->right->left = node->left;
      node->left->right = node->right;
      if (base == node) {
        base = base->right;
      }
    }
  }

  void consolidate() {
    int size = 0;
    Node *node = root;
    do {
      Node *next = node->right;
      Node *x = node;
      while (x->degree < size && aux[x->degree] != nullptr) {
        Node *y = aux[x->degree];
        aux[x->degree] = nullptr;
        if (x->second < y->second) {
          link(y, x);
        } else {
          link(x, y);
          x = y;
        }
      }
      if (size <= x->degree) {
        size = x->degree + 1;
      }
      aux[x->degree] = x;
      node = next;
    } while (node != root);
    root = nullptr;
    for (int d = 0; d < size; d++) {
      if (aux[d] != nullptr) {
        insert(aux[d]);
        aux[d] = nullptr;
      }
    }
  }

  void cut(Node *node) {
    while (node != nullptr) {
      unlink(node);
      insert(node);
      Node *parent = node->parent;
      node = nullptr;
      if (parent->parent != nullptr) {
        if (parent->mark) {
          node = parent;
        } else {
          parent->mark = true;
        }
      }
    }
  }

public:
  FibonacciHeap(int n) : root(nullptr), index(n, nullptr), aux(n, nullptr) {}

  bool empty() { return root == nullptr; }

  bool contains(int v) { return index[v] != nullptr; }

  int extract_min() {
    Node *min = root;
    unlink(min);
    if (Node *node = min->child; node != nullptr) {
      do {
        Node *next = node->right;
        insert(node);
        node = next;
      } while (node != min->child);
    }
    if (root != nullptr && root != root->right) {
      consolidate();
    }
    int v = min->first;
    delete min;
    index[v] = nullptr;
    return v;
  }

  void insert(int v, double d) { insert(new Node(v, d)); }

  void decrease_key(int v, double d) {
    Node *node = index[v];
    node->second = d;
    if (node->parent != nullptr && node->second < node->parent->second) {
      cut(node);
    }
  }
};

public:

  std::vector<std::vector<std::pair<int, double>>> A;

  Graph(int n) : n(n), E(), A(n) {}

  void add_edge(int u, int v, double w) {
    E.emplace_back(u, v, w);
    A[u].emplace_back(v, w);

    E.clear();
