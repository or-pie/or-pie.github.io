Floyd-Warshall 算法

Floyd-Warshall 算法是一种解决包含负权重边的有向图上的全节点对最短路径问题的算法。

1. 引言

给定有向图 \(G = (V, E)\) 和权重函数 \(w : E \rightarrow \mathbb{R}\)，我们希望找到对于每一组节点对 \((i, j) \in V^2\)，从节点 \(i\) 到节点 \(j\) 的最短路径，即权重最小的路径。我们可以通过运行 \(|V|\) 次单源点最短路径算法，每一次使用一个不同的节点作为源点，来解决全节点对最短路径问题。如果所有的边权重都是非负值，那么可以使用 Dijkstra 算法，总运行时间是 \(O(|V|^3)\)（线性表实现）、\(O(|V| |E| \log |V|)\)（二叉堆实现）或 \(O(|V|^2 \log |V| + |V| |E|)\)（斐波那契堆实现）。如果输入图包含负权重边，那么就不能使用 Dikstra 算法，只能必须运行效率更低的 Bellman-Ford 算法，每次使用一个不同的节点作为源点，总运行时间是 \(O(|V|^2 |E|)\)。

在本文，我们介绍解决包含负权重边的有向图上的全节点对最短路径问题的 Floyd-Warshall 算法，其运行时间为 \(O(|V|^3)\)。

2. 算法思想

在下面的讨论中，我们允许输入图中包含负权重边，但暂时先假定不存在负权重环。Floyd-Warshall 算法依赖于下面的观察。

最短路径的中间节点

假定有向图 \(G\) 的所有节点为 \(V = \{0, 1, \dots, n - 1\}\)，考虑由前 \(k + 1\) 个节点（\(0 \le k < n\)）组成的子集 \(\{0, 1, \dots, k\}\)。对于任意节点对 \((i, j) \in V^2\)，考虑从节点 \(i\) 到节点 \(j\) 的所有中间节点均取自集合 \(\{0, 1, \dots, k\}\) 的路径，并且设 \(p\) 为其中权重最小的路径。

  • 如果节点 \(k\) 不是路径 \(p\) 的中间节点，则路径 \(p\) 上的所有中间节点都属于集合 \(\{0, 1, \dots, k - 1\}\)。因此，从节点 \(i\) 到节点 \(j\) 的中间节点取自集合 \(\{0, 1, \dots, k - 1\}\) 的一条最短路径也是从节点 \(i\) 到节点 \(j\) 的中间节点取自集合 \(\{0, 1, \dots, k\}\) 的一条最短路径；
  • 如果节点 \(k\) 是路径 \(p\) 的中间节点，则将路径 \(p\) 分解为 \(\langle i \stackrel{p_1}{\rightsquigarrow} k \stackrel{p_2}{\rightsquigarrow} j\rangle\)。根据最短路径问题的最优子结构，\(p_1\) 是从节点 \(i\) 到节点 \(k\) 的中间节点取自集合 \(\{0, 1, \dots, k - 1\}\) 的一条最短路径。类似地，\(p_2\) 是从节点 \(k\) 到节点 \(j\) 的中间节点取自集合 \(\{0, 1, \dots, k - 1\}\) 的一条最短路径。

基于上面的观察，我们定义如下全节点对最短路径问题的 动态规划模型。

动态规划模型

对 \(k \in \{0, 1, \dots, n\}\) 和 \((i, j) \in V^2\)，定义距离矩阵 \(D^{(k)} = [d^{(k)}_{i,j}]_{(i,j)\in V^2}\)，其中 \(d^{(k)}_{i,j}\) 为从节点 \(i\) 到节点 \(j\) 的中间节点取自集合 \(\{0, 1, \dots, k - 1\}\) 的最短路径权重。具体来说，当 \(k = 0\) 时，从节点 \(i\) 到节点 \(j\) 不包含任何中间节点的路径最多只有一条边，因此有

\[ d^{(0)}_{i, j} = \begin{cases} 0, & \text{if $i = j$}; \\ w(i, j), & \text{if $i \neq j$ and $(i, j) \in E$}; \\ \infty, & \text{if $i \neq j$ and $(i, j) \not\in E$}. \end{cases} \]

对于 \(k = 0, 1, \dots, n - 1\)，按下面的公式递推计算 \(d^{(k + 1)}_{i,j}\)：

\[ d^{(k + 1)}_{i, j} = \begin{cases} d^{(k)}_{i, j}, & \text{if $d^{(k)}_{i,j} \le d^{(k)}_{i,k} + d^{(k)}_{k,j}$}; \\ d^{(k)}_{i,k} + d^{(k)}_{k,j}, & \text{if $d^{(k)}_{i,j} > d^{(k)}_{i,k} + d^{(k)}_{k,j}$}.\end{cases} \]

对于任何路径来说，所有的中间节点都属于集合 \(V = \{0, 1, \dots, n - 1\}\)，因此 \(d^{(n)}_{i, j}\) 就是从节点 \(i\) 到节点 \(j\) 的最短路径权重，即对于所有的节点对 \((i, j) \in V^2\)，有 \(d^{(n)}_{i, j} = \delta(i, j)\)。

另外，我们在计算距离矩阵 \(D^{(k)}\) 时同时记录前驱矩阵 \(\Pi^{(k)} = [\pi^{(k)}_{i,j}]_{(i,j)\in V^2}\)，其中 \(\pi^{(k)}_{i, j}\) 表示 节点 \(j\) 在该最短路径上的前驱节点。具体来说，当 \(k = 0\) 时，从节点 \(i\) 到节点 \(j\) 的最短路径没有中间节点，因此

\[ \pi^{(0)}_{i, j} = \begin{cases} nil, & \text{if $i = j$ or $(i, j) \not\in E$}; \\ i, & \text{if $i \neq j$ and $(i, j) \in E$}. \end{cases} \]

对于 \(k = 0, 1, \dots, n - 1\)，如果节点 \(k\) 不是从节点 \(i\) 到节点 \(j\) 的中间节点取自集合 \(\{0, 1, \dots, k\}\) 的一条最短路径的中间节点，那么 \(\pi^{(k + 1)}_{i, j}\) 和 \(\pi^{(k)}_{i, j}\) 是一样的。否则，\(\pi^{(k + 1)}_{i, j}\) 和 \(\pi^{(k)}_{k, j}\) 是一样的。也就是说，对于 \(k = 0, 1, \dots, n - 1\)，

\[ \pi^{(k + 1)}_{i, j} = \begin{cases} \pi^{(k)}_{i, j}, & \text{if $d^{(k)}_{i,j} \le d^{(k)}_{i,k} + d^{(k)}_{k,j}$}; \\ \pi^{(k)}_{k, j}, & \text{if $d^{(k)}_{i,j} > d^{(k)}_{i,k} + d^{(k)}_{k,j}$}. \end{cases} \]

在下面的算法中，第 2–16 行设置初始的距离矩阵 \(D^{(0)}\) 和前驱矩阵 \(\Pi^{(0)}\)，第 17–29 行使用三层循环依次计算距离矩阵 \(D^{(1)}, \dots, D^{(n)}\) 和前驱矩阵 \(\Pi^{(1)}, \dots, \Pi^{(n)}\)。

void floyd_warshall(const Graph &G) {
  dist.assign(G.n + 1, std::vector<std::vector<double>>(
              G.n, std::vector<double>(G.n, inf)));
  prev.assign(G.n + 1, std::vector<std::vector<int>>(
              G.n, std::vector<int>(G.n, nil)));
  for (int i = 0; i < G.n; i++) {
    dist[0][i][i] = 0;
  }
  for (auto [i, j, w] : G.E) {
    if (i != j) {
      dist[0][i][j] = w;
      prev[0][i][j] = i;
    }
  }
  for (int k = 0; k < G.n; k++) {
    for (int i = 0; i < G.n; i++) {
      for (int j = 0; j < G.n; j++) {
        if (dist[k][i][j] <= dist[k][i][k] + dist[k][k][j]) {
          dist[k + 1][i][j] = dist[k][i][j];
          prev[k + 1][i][j] = prev[k][i][j];
        } else {
          dist[k + 1][i][j] = dist[k][i][k] + dist[k][k][j];
          prev[k + 1][i][j] = prev[k][k][j];
        }
      }
    }
  }
}

代码说明

在代码中，我们使用 dist[k][i][j] 和 prev[k][i][j] 表示 \(d^{(k)}_{i, j}\) 和 \(\pi^{(k)}_{i, j}\)。

3. 具体实现

我们可以对 Floyd-Warshall 算法进行空间优化，使其只占用 \(O(|V|^2)\) 的空间。方法是省去 \(D^{(k)}\) 和 \(\Pi^{(k)}\) 的上标，在算法运行过程中直接在初始的二维距离矩阵 \(D\) 和前驱矩阵 \(\Pi\) 上进行修改。在算法的第 \(k\) 轮（\(k = 0, 1, \dots, n - 1\)）迭代中，对 \(d_{i,j}\) 和 \(d_{i,k} + d_{k,j}\) 进行比较时很可能 \(d_{i,k}\) 和 \(d_{k,j}\) 已经在本轮迭代中被更新了，即 \(d_{i,k}\) 和 \(d_{k,j}\) 对应的最短路径很可能已经使用了节点 \(k\) 作为中间节点了。即使 \(d_{i,k}\) 或 \(d_{k,j}\) 对应的最短路径已经使用了节点 \(k\) 作为中间节点，除非存在负权重环，否则一定有 \(d_{i,k} \le d_{i,k} + d_{k,j}\)，不会影响算法的正确性。

若要检测负权重环，需要在 Floyd-Warshall 算法的运行过程中观察距离矩阵的对角线上是否有负值。如果距离矩阵的对角线上的某个值 \(d_{i,i} < 0\)，那么输入图中存在包含节点 \(i\) 的负权重环，此时算法可以立刻结束并报告图中存在负权重环。特别地，在初始化距离矩阵和前驱矩阵时，也应当检查对角线上的值，以此检测输入图中是否存在权重为负值的 自环。我们对 Floyd-Warshall 算法进行修改，使其返回一个布尔值以表明输入图中是否存在负权重环。如果存在负权重环，算法将返回 false，并且用变量 clue 记录该负权重环中的某个节点。否则，算法将返回 true，并且给出所有节点对之间的最短路径和它们的权重。

bool floyd_warshall(const Graph &G) {
  dist.assign(G.n, std::vector<double>(G.n, inf));
  prev.assign(G.n, std::vector<int>(G.n, nil));
  for (int i = 0; i < G.n; i++) {
    dist[i][i] = 0;
  }
  for (auto [i, j, w] : G.E) {
    if (dist[i][j] > w) {
      dist[i][j] = w;
      prev[i][j] = i;
      if (dist[i][i] < 0) {
        clue = i;
        return false;
      }
    }
  }
  for (int k = 0; k < G.n; k++) {
    for (int i = 0; i < G.n; i++) {
      for (int j = 0; j < G.n; j++) {
        if (dist[i][j] > dist[i][k] + dist[k][j]) {
          dist[i][j] = dist[i][k] + dist[k][j];
          prev[i][j] = prev[k][j];
          if (dist[i][i] < 0) {
            clue = i;
            return false;
          }
        }
      }
    }
  }
  clue = nil;
  return true;
}

#include "floyd-warshall.h"

int main() {
  Graph G(5);
  G.add_edge(0, 1, 3);
  G.add_edge(0, 2, 8);
  G.add_edge(0, 4, -4);
  G.add_edge(1, 3, 1);
  G.add_edge(1, 4, 7);
  G.add_edge(2, 1, 4);
  G.add_edge(3, 0, 2);
  G.add_edge(3, 2, -5);
  G.add_edge(4, 3, 6);

  FloydWarshall FW;
  if (FW.floyd_warshall(G)) {
    std::cout << "距离矩阵：" << std::endl;
    for (int i = 0; i < G.n; i++) {
      for (int j = 0; j < G.n; j++) {
        std::cout << std::setw(4) << FW.dist[i][j];
      }
      std::cout << std::endl;
    }
    std::cout << "前驱矩阵：" << std::endl;
    for (int i = 0; i < G.n; i++) {
      for (int j = 0; j < G.n; j++) {
        std::cout << std::setw(4) << FW.prev[i][j];
      }
      std::cout << std::endl;
    }
  } else {
    std::cout << "存在负权重环：";
    std::vector<int> c = FW.reconstruct_negative_weight_cycle();
    for (int i = 0; i < c.size(); i++) {
      std::cout << c[i] << (i < c.size() - 1 ? "->" : "");
    }
    std::cout << std::endl;
  }

  return 0;
}

#ifndef FLOYD_WARSHALL_H
#define FLOYD_WARSHALL_H

#include "graph.h"

class FloydWarshall {
public:
  std::vector<std::vector<double>> dist;
  std::vector<std::vector<int>> prev;
  int clue;

  bool floyd_warshall(const Graph &G) {
    dist.assign(G.n, std::vector<double>(G.n, inf));
    prev.assign(G.n, std::vector<int>(G.n, nil));
    for (int i = 0; i < G.n; i++) {
      dist[i][i] = 0;
    }
    for (auto [i, j, w] : G.E) {
      if (dist[i][j] > w) {
        dist[i][j] = w;
        prev[i][j] = i;
        if (dist[i][i] < 0) {
          clue = i;
          return false;
        }
      }
    }
    for (int k = 0; k < G.n; k++) {
      for (int i = 0; i < G.n; i++) {
        for (int j = 0; j < G.n; j++) {
          if (dist[i][j] > dist[i][k] + dist[k][j]) {
            dist[i][j] = dist[i][k] + dist[k][j];
            prev[i][j] = prev[k][j];
            if (dist[i][i] < 0) {
              clue = i;
              return false;
            }
          }
        }
      }
    }
    clue = nil;
    return true;
  }

  std::vector<int> reconstruct_shortest_path(int u, int v) {
    if (clue != nil || dist[u][v] == inf) {
      return {};
    } else {
      std::vector<int> p;
      for (int x = v; x != nil; x = prev[u][x]) {
        p.push_back(x);
      }
      std::reverse(p.begin(), p.end());
      return p;
    }
  }

  std::vector<int> reconstruct_negative_weight_cycle() {
    if (clue == nil) {
      return {};
    } else {
      std::vector<bool> aux(prev.size(), false);
      std::vector<int> p;
      int x = clue;
      while (not aux[x]) {
        aux[x] = true;
        p.push_back(x);
        x = prev[clue][x];
      }
      std::vector<int> c = {x};
      for (auto it = p.rbegin(); *it != x; it++) {
        c.push_back(*it);
      }
      c.push_back(x);
      return c;
    }
  }
};

#endif

#ifndef GRAPH_H
#define GRAPH_H

#include <bits/stdc++.h>

class Graph {
public:
  int n;
  std::vector<std::tuple<int, int, double>> E;
  std::vector<std::vector<std::pair<int, double>>> A;

  Graph(int n) : n(n), E(), A(n) {}

  void add_edge(int u, int v, double w) {
    E.emplace_back(u, v, w);
    A[u].emplace_back(v, w);
  }

  void clear() {
    E.clear();
    A.assign(n, {});
  }
};

const double inf = std::numeric_limits<double>::infinity();
const int nil = -1;

#endif
